# BIDSafe
APSS2BIDS — A self-contained desktop tool (tkinter + dcm2niix) that converts clinical DICOM archives into BIDS-compliant datasets. Features automatic modality detection, contrast-agent labelling (ce- entity), incremental processing of new sessions, and a privacy-preserving confidential mapping for subject pseudonymization

# GUI
A very simple GUI. Fewer options, more functions under the hood: the app is designed to ease the surgical staff workload.
